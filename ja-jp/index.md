# zp3n's Toybox
自分の制作物の名称と概要です。  

## Promptly MOD (for Minecraft)
マイクラサーバーの長くて打つのが面倒なサーバーコマンドを、自分で短いコマンドに置き換えることができます。
またSkyblock用に破壊・攻撃をトグルとホールドにコマンド１つで切り替えられる機能もあります。  

**[Promptly Repository](https://github.com/zp3n/Promptly)**

## MapRouter MOD (for Minecraft)
クライアント側で空間に道標となる線を描画するMODです。
コンフィグで座標を設定できます。  

**[MapRouter Repository](https://github.com/zp3n/MapRouter)**
