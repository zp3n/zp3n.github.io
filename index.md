# zp3n's Toybox
Here is the name and overview of the product I created.

## Promptly MOD (for Minecraft)
A mod for Minecraft servers that allows you to replace long server commands with shorter ones of your choice.  
There is also a feature that allows you to switch Toggle/Hold modes for Attack/Break using a command.  

**[Promptly Repository](https://github.com/zp3n/Promptly)**

## MapRouter MOD (for Minecraft)
This is a mod that draws guide lines in the game world on the client side.  
You specify the coordinates in the config file.

**[MapRouter Repository](https://github.com/zp3n/MapRouter)**
